# Validation Checklist
## Quality Control for Candidate Control Skill Outputs

This checklist ensures all outputs meet quality standards before being shared with Account Managers or clients.

---

## PRE-SUBMISSION VALIDATION

Use this checklist before sending any communication. Every item should be verified as complete and accurate.

---

## PHASE 1: CLABVISSS INTAKE VALIDATION

### Data Completeness Check

**Commitment & Flexibility:**
- [ ] Current interview status documented
- [ ] Willingness to move quickly assessed
- [ ] Deadline extension flexibility confirmed
- [ ] Decision-making approach understood
- [ ] Commitment level rated (High/Medium/Low)

**Location:**
- [ ] Current location documented
- [ ] Relocation willingness confirmed
- [ ] Remote/hybrid expectations clear
- [ ] Commute tolerance established
- [ ] Any location deal-breakers identified

**Availability & Timeline:**
- [ ] All competing processes listed with company names
- [ ] Current stage in each process documented
- [ ] Expected offer dates from competitors recorded
- [ ] Fastest-moving competitor identified
- [ ] Ideal start date documented
- [ ] Notice period length confirmed

**Background:**
- [ ] Employment gaps explained (if any)
- [ ] Reference availability confirmed
- [ ] Any performance issues disclosed
- [ ] Credentials verified
- [ ] Non-compete status clarified

**Visa Status:**
- [ ] Current authorization status documented
- [ ] Sponsorship needs identified
- [ ] Expiration dates recorded (if applicable)
- [ ] Transfer timeline understood
- [ ] Any complications noted

**Ideal vs. Acceptable:**
- [ ] Ideal role described
- [ ] Must-have requirements listed
- [ ] Nice-to-have preferences noted
- [ ] Acceptable minimum defined
- [ ] Deal-breakers identified

**Salary & Compensation:**
- [ ] Current base salary documented
- [ ] Current bonus documented
- [ ] Current equity value documented
- [ ] Total current compensation calculated
- [ ] Target compensation range documented
- [ ] Minimum acceptable offer identified
- [ ] Compensation motivations understood

**Start Date Flexibility:**
- [ ] Standard notice period documented
- [ ] Earliest possible start date confirmed
- [ ] Flexibility level assessed
- [ ] Obstacles to quick start identified

**Stage in Other Processes:**
- [ ] All competing companies listed
- [ ] Stage in each process documented
- [ ] Current ranking of opportunities noted
- [ ] Our position in ranking confirmed
- [ ] Decision factors understood

### Quality Check

- [ ] **No vague answers** - All responses are specific and detailed
- [ ] **Probing completed** - Follow-up questions were asked where needed
- [ ] **Red flags documented** - Any concerns are clearly noted
- [ ] **Quotes captured** - Verbatim language recorded for motivations/selling points
- [ ] **Recruiter confidence** - Feel good about presenting this candidate

---

## PHASE 2: CV SUBMISSION VALIDATION

### BLACK TEXT (Client-Facing) Check

**Subject Line:**
- [ ] Includes candidate full name
- [ ] Includes years of experience
- [ ] Includes specific experience type
- [ ] Under 65 characters
- [ ] Professional format

**Content Structure:**
- [ ] Paragraph 1: Clear intro with years and selling points
- [ ] Paragraph 2: Relevant background and achievements
- [ ] Paragraph 3: Motivations and specific interest
- [ ] Paragraph 4 (if needed): Additional context or closing
- [ ] Total length: 300-400 words

**Content Quality:**
- [ ] **No bullet points** - Full paragraphs only
- [ ] **Specific achievements** - Numbers and metrics included
- [ ] **Tailored to role** - Selling points match client needs
- [ ] **Motivation included** - Why leaving current role
- [ ] **Client-specific interest** - Why THIS company
- [ ] **Positive framing** - No negative information
- [ ] **Professional tone** - Confident and warm
- [ ] **Copy-paste ready** - AM doesn't need to edit anything

**Content Accuracy:**
- [ ] Candidate name spelled correctly
- [ ] Current company name accurate
- [ ] Job titles accurate
- [ ] Years of experience correct
- [ ] Numbers/metrics verified
- [ ] No placeholder [TBD] text

---

### RED TEXT (Internal Notes) Check

**Required Sections Present:**
- [ ] Visa Status section
- [ ] Compensation section
- [ ] Competing Processes section
- [ ] Red Flags / Concerns section
- [ ] Title Expectations section
- [ ] Location / Work Setup section (if relevant)
- [ ] Start Date section
- [ ] Additional Context section (if needed)
- [ ] Closing Strategy section

**Visa Status:**
- [ ] Current status documented
- [ ] Sponsorship requirements clear
- [ ] Any complications noted
- [ ] Risk level assessed

**Compensation:**
- [ ] Current base documented with $
- [ ] Current bonus documented with $
- [ ] Current equity documented with $
- [ ] Total current comp calculated
- [ ] Target compensation range documented
- [ ] Minimum documented
- [ ] Context about flexibility included
- [ ] Alignment with client budget noted

**Competing Processes:**
- [ ] All companies listed by name
- [ ] Stage in each process specified
- [ ] Expected offer dates documented
- [ ] Current ranking noted
- [ ] Urgency level indicated

**Red Flags / Concerns:**
- [ ] All concerns from intake documented
- [ ] Severity level clear
- [ ] Deal-breakers identified
- [ ] Salvageable vs. unsalvageable noted

**Title Expectations:**
- [ ] Required title documented
- [ ] Unacceptable titles noted
- [ ] Context provided

**Start Date:**
- [ ] Notice period documented
- [ ] Earliest start date specified
- [ ] Any constraints noted
- [ ] Flexibility level indicated

**Closing Strategy:**
- [ ] Specific recommendations provided
- [ ] Timeline for action noted
- [ ] What to emphasize listed
- [ ] How to address concerns outlined

---

### Format Check

- [ ] **Clear separation** - BLACK and RED text visibly distinct
- [ ] **🔴 emoji used** - Internal section clearly marked
- [ ] **"DO NOT SEND TO CLIENT" warning** - Present in RED section
- [ ] **Proper formatting** - Clean, professional appearance
- [ ] **No formatting errors** - No weird spacing or characters

## FINAL VALIDATION QUESTIONS

Before sending any output, answer these questions:

### For CV Submissions:
1. **Could the AM copy and paste the BLACK TEXT directly to the client without any editing?**
   - [ ] Yes → Good to go
   - [ ] No → Revise BLACK TEXT

2. **Are all sensitive details properly flagged in RED TEXT?**
   - [ ] Yes → Good to go
   - [ ] No → Add to RED TEXT

3. **Would the client feel excited and confident about this candidate after reading?**
   - [ ] Yes → Good to go
   - [ ] No → Strengthen selling points

4. **Does the AM have everything they need to strategize and close?**
   - [ ] Yes → Good to go
   - [ ] No → Add to Closing Strategy

---

### For Client Feedback:
1. **Does the client understand the candidate is still interested?**
   - [ ] Yes → Good to go
   - [ ] No → Strengthen interest statements

2. **Does the client know where they stand vs. competitors?**
   - [ ] Yes → Good to go
   - [ ] No → Clarify ranking

3. **Are all concerns properly flagged internally but not shared with client?**
   - [ ] Yes → Good to go
   - [ ] No → Review separation of BLACK/RED text

4. **Does the AM have a clear closing strategy?**
   - [ ] Yes → Good to go
   - [ ] No → Develop strategy section

## COMMON MISTAKES TO CATCH

### Content Mistakes
- [ ] Vague language ("good experience," "nice person")
- [ ] Negative information in client-facing text
- [ ] Missing specific numbers/metrics
- [ ] Generic, non-tailored descriptions
- [ ] Forgetting to mention motivation for leaving
- [ ] Not specifying why interested in THIS company
- [ ] Including RED TEXT information in BLACK TEXT

### Format Mistakes
- [ ] Bullet points in client-facing emails
- [ ] RED TEXT not clearly separated
- [ ] Missing 🔴 emoji or warning label
- [ ] Inconsistent formatting
- [ ] Run-on paragraphs
- [ ] Poor spacing

### Data Mistakes
- [ ] Incomplete CLABVISSS data
- [ ] Missing compensation breakdowns
- [ ] Vague competing process information
- [ ] Unclear timelines
- [ ] Ambiguous interest level
- [ ] Missing red flags

### Tone Mistakes
- [ ] Too casual or informal
- [ ] Too stiff or robotic
- [ ] Overly salesy or hyperbolic
- [ ] Not confident enough
- [ ] Hedging too much

## APPROVAL CHECKLIST

Before considering output complete and ready to send:

**All validations completed:**
- [ ] Relevant phase checklist completed (CLABVISSS, CV, RATESSS, or Feedback)
- [ ] Universal checks completed
- [ ] Final validation questions answered
- [ ] Common mistakes reviewed and corrected

**Quality confirmed:**
- [ ] Data is complete and accurate
- [ ] Format is correct and professional
- [ ] Tone is appropriate
- [ ] Content is specific and compelling
- [ ] Separation of client/internal info is clear

**Ready for action:**
- [ ] AM can immediately use this output
- [ ] No follow-up questions needed
- [ ] All information necessary for next steps is present
- [ ] Closing strategy is clear and actionable
