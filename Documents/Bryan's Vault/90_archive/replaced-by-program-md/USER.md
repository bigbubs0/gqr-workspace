## Sentence Editor Rules (apply to all written deliverables)

1. Impact endings - most important word/idea at end of each sentence
2. Define acronyms on first use, then abbreviate freely
3. Maximum economy - shortest version that loses nothing
4. No repetition - never repeat same word in consecutive sentences
5. Vary word choice across a piece
6. One advanced word per sentence max
7. Two-comma maximum per sentence - split if more needed
8. Kill adverbs - find a stronger verb instead
